Two trends:
1. With gmem, consistent, monotonic increase in BW with increasing size; With host, inflection point and decline in BW as size increases

2. With gmem, using double (64-bit words) doubles performance. With host, slight improvement for small data sizes, and then actually decline in performance.

3. Predictability: Note the relative SD for host.
